## Installation

### Recover the project

Get the project from github using:

- https:

```sh
git clone https://github.com/Ydays-Ynuit/front.git
```

or

- ssh:

```sh
git clone git@github.com:Ydays-Ynuit/front.git
```

Then, enter in the project folder

### Install dependencies

```sh
yarn
```
